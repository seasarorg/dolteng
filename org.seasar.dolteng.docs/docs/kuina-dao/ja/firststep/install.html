インストール

Kuina-Daoを簡単に使用するためのEclipseプラグイン「Dolteng」「Diigu」「DbLauncher」をインストールします．

  • Eclipseのメニューから「Help」－「Software Updates」－「Find and Install」を選択してください．
  • 「Install/Updates」ウィザードが開きます．
  • 「Search for new features to install」を選択してください．
  • 「Next」ボタンを押下してください．
  • 「Install」ページが開きます．
  • 「New Remote Site」ボタンを押下してください．
  • 「New Update Site」ダイアログが開きます．
  • 「Name」に"Seasar Project Updates 3.1"と入力してください．
  • 「URL」に"http://eclipse.seasar.org/updates/3.1/"と入力してください．
  • 「OK」ボタンを押下してください．
  • 「Install」ページに戻ります．
  • 続けて「New Remote Site」ボタンを押下してください．
  • 「New Update Site」ダイアログが開きます．
  • 「Name」に"Seasar Project Updates 3.2"と入力してください．
  • 「URL」に"http://eclipse.seasar.org/updates/3.2/"と入力してください．
  • 「OK」ボタンを押下してください．
  • 「Install」ページに戻ります．
  • 追加されたSeasarプロジェクトのアップデートサイトがチェック状態になっていることを確認してください．
  • 「Finish」ボタンを押下してください．
  • 「Updates」ページが開きます．
  • 「Seasar Project Updates 3.1」の下の「Diigu」をチェックしてください．
  • 「Seasar Project Updates 3.2」の下の「DbLauncher」および「Dolteng」をチェックしてください．
  • 「Next」ボタンを押下してください．
  • 「Install」ページが開きます．
  • ライセンスを確認して「I accept the terms in the license agreement」を選択してください．
  • 「Next」ボタンを押下してください．
  • 「Install」ページが開きます．
  • 「Finish」ボタンを押下してください．
  • 「Verification」ページが開きます．
  • 「Install All」ボタンを押下してください．
  • 「Install/Update」ダイアログが開きます．
  • 「Yes」ボタンを押下してください．

Eclipseが再起動してインストールは終了です． 「スタンドアロン編 」へ進んでください．
