Welcome

Kuina-Dao は，JavaEE 5 の O/R マッピング標準，Java Persistence API (JPA) 上で利用可能な Data Access Object (DAO) フレームワークです． Kuina-Dao を利用することにより，Java インタフェースを定義するだけで動的な問い合わせを作成したり，名前付き問い合わせを容易に呼び出したりすることが可能になります．
