Teeda Core

Teeda Coreとは？

Teeda Coreとは、JavaServer Facesの実装で、主に以下のような機能を提供します。

  • JSFの標準コンポーネント(JSFに規定されているコンポーネント一式)
  • S2ContainerによるDIとAOPの機能
  • JSF用テスティングフレームワーク(TeedaUnit)
現状はJSF1.1に準拠した実装となっています。
