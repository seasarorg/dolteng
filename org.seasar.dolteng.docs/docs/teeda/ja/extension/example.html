セットアップ

事前条件

Teeda HTML Exampleの配布ファイルは， EclipseのTomcsatプロジェクトをアーカイブしたものになっています．

事前にEclipseを用意して， Sysdeo/SQL Tomcat Launcherプラグインをセットアップしてください． 手順は こちら ．

ダウンロード

こちら からTeeda HTML Exampleをダウンロードしてください．

インポート

以下の手順でTeeda HTML ExampleをEclipseワークスペースにインポートしてください．

  • ダウンロードしたTeeda HTML Exampleの配布ファイルをEclipseのワークスペースまたは適当なディレクトリに解凍してください．
  • Eclipseの「File」－「Import」メニューで「Import Wizard」を開きます．
  • 「Select a import source」で「General」－「Existing Project into Workspace」を選択し， 「Next」ボタンを押下します．
  • 「Select a root directory」でTeeda HTML Exampleの配布ファイルを解凍したディレクトリを指定します．
  • 「Projects」で "teeda-html-example"が チェックされていることを確認して「Finish」ボタンを押下します．
  • インポートされたteeda-html-exampleプロジェクトを右クリックして， 「Tomcat」－「コンテキスト定義の更新」を選択します．

サンプル実行

Tomcatの起動

メニューの「Tomcat」－「Tomcat起動」を選択またはツールバーの「Tomcat起動」ボタンを押下して， Tomcatを起動します

画面の表示

Tomcatがローカルホストのポート番号8080で動作している場合， ブラウザで以下のURLにアクセスすると Teeda HTML Exampleの開始画面が表示されます．

  • http://localhost:8080/teeda-html-example/
