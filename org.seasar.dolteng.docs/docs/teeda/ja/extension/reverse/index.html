Teeda Extension 逆引きリファレンス

Teeda Extensionの使い方を「～をするには」という目的から逆引き形式で探せるドキュメントです．

出力
HTMLのボディにテキストを出力する方法や， 要素の属性， ヘッダのタイトルを動的に変更する方法を紹介します．
リンク
リンクを作成する方法や， パラメータを動的に変更する方法を紹介します．
入力フォーム
テキストフィールやラジオボタン， チェックボックス等， 入力フォームの使い方を説明します．
バリデーション
フォームから入力されたリクエストパラメータをチェックする方法を説明します．
コンバート
ページクラスが持つ値と， HTMLにレンダリングされたりフォームから入力されたりする値を， 相互変換する方法を説明します．
繰り返し
リストの項目やテーブルの行・列を繰り返し表示する方法や， その中で入力項目を扱う方法を説明します．
条件制御
実行時の条件によって画面の一部を表示したり非表示にしたりする方法を説明します．
レイアウト
レイアウトを使う方法を説明します．
画面遷移と状態
画面遷移する方法と状態の引き継ぎを制御する方法を説明します．
プレビュー
テンプレートHTMLをブラウザで直接プレビューする際に便利な機能を説明します．
