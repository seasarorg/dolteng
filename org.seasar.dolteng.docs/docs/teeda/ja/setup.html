Eclipseでの利用

EclipseでTeedaを利用するには，Eclipseプラグイン「Dolteng」「DbLauncher」「 Sysdeo/SQLI Tomcat Launcher 」を使用するのが便利です．

Eclipseプラグインのインストール

以下の手順でEclipseプラグインをインストールしてください．

  • Eclipseのメニューから「Help」－「Software Updates」－「Find and Install」を選択してください．
  • 「Install/Updates」ウィザードが開きます．
  • 「Search for new features to install」を選択してください．
  • 「Next」ボタンを押下してください．
  • 「Install」ページが開きます．
  • 「New Remote Site」ボタンを押下してください．
  • 「New Update Site」ダイアログが開きます．
  • 「Name」に" Seasar Project Updates 3.2 "と入力してください．
  • 「URL」に" http://eclipse.seasar.org/updates/3.2/ "と入力してください．
  • 「OK」ボタンを押下してください．
  • 「Install」ページに戻ります．
  • 「New Remote Site」ボタンを押下してください．
  • 「New Update Site」ダイアログが開きます．
  • 「Name」に" Seasar Project Updates 3.3 "と入力してください．
  • 「URL」に" http://eclipse.seasar.org/updates/3.3/ "と入力してください．
  • 「OK」ボタンを押下してください．
  • 「Install」ページに戻ります．
  • 追加されたSeasarプロジェクトのアップデートサイトがチェック状態になっていることを確認してください．
  • 「Finish」ボタンを押下してください．
  • 「Updates」ページが開きます．
  • 「Seasar Project Updates 3.2」の下の以下のフィーチャーをチェックしてください．
    • DbLauncher (任意)
    • Sysdeo/SQLI Tomcat Launcher (任意)
  • 「Seasar Project Updates 3.3」の下の以下のフィーチャーをチェックしてください．
    • Dolteng
  • 「Next」ボタンを押下してください．
  • 「Install」ページが開きます．
  • ライセンスを確認して「I accept the terms in the license agreement」を選択してください．
  • 「Next」ボタンを押下してください．
  • 「Install」ページが開きます．
  • 「Finish」ボタンを押下してください．
  • 「Verification」ページが開きます．
  • 「Install All」ボタンを押下してください．
  • 「Install/Update」ダイアログが開きます．
  • 「Yes」ボタンを押下してください．

Eclipseが再起動するとインストールは終了です．

Tomcat Launcerプラグインの設定

Sysdeo/SQLI Tomcat Launcherプラグインをインストールした場合は， 以下の手順で設定を行います． なお，事前にTomcatのインストールが必要です．Tomcatは こちら からダウンロードしてください．

  • Eclipseのメニューから「Window」－「Preferences」を選択してください．
  • 「Preferences」ダイアログが開きます．
  • 左のペインで「Tomcat」を選択してください．
  • 「Tomcatバージョン」で使用するTomcatのバージョンを選択してください．
  • 「Tomcatホーム」にTomcatをインストールしたディレクトリを指定してください．
  • 左のペインで「Tomcat」－「JVMの設定」を選択してください．
  • 「JRE]でTomcatの実行に使用するJREを選択してください．
  • 「OK」ボタンを押下してください．
